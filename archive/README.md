# Archived Files

This directory contains archived versions of the software that are no longer part of the main release.

## Free Version

`main_tkinter_free.py` - The original free version that used Windows WASAPI loopback for audio capture without requiring third-party software.

**Note**: This version is archived and no longer maintained. The current commercial release (v1.0.0) includes all features from the free version plus additional improvements and support.
