# Feature List - Real-Time Game Voice Translation

## Core Features

### Real-Time Translation
- ✅ **Bidirectional Translation**:
  - ✅ Teammate → You: Translates teammates' voices to your language
  - ✅ You → Teammates: Translates your voice to teammates' languages
- ✅ Captures in-game voice chat from competitive multiplayer games
- ✅ Captures your microphone input for voice translation
- ✅ Transcribes speech using OpenAI Whisper AI
- ✅ Automatic language detection
- ✅ Smart language tracking for auto-detecting teammate languages
- ✅ Translates to 15+ languages (English, Russian, Spanish, German, French, Portuguese, Polish, Turkish, Chinese, Japanese, Korean, Ukrainian, Italian, Arabic, Hindi)
- ✅ Low latency (<1 second with local models, faster than API)
- ✅ **Offline translation** - Works without internet connection (no API keys needed)
- ✅ Local AI models for privacy and speed

### Display Features
- ✅ On-screen subtitle overlay
- ✅ Customizable subtitle size (12-48px)
- ✅ Adjustable subtitle position
- ✅ Always-on-top display
- ✅ Auto-hide when no text
- ✅ Modern, clean design

### Audio Features
- ✅ Multiple audio capture methods
  - VB-Audio Virtual Cable (recommended)
  - Windows Stereo Mix
  - WASAPI loopback
- ✅ Automatic audio device detection
- ✅ Audio level monitoring
- ✅ Optional text-to-speech playback
- ✅ **Virtual Microphone Output**: Routes translated voice to VB-Audio Cable Input for game compatibility
- ✅ **Microphone Capture**: Separate microphone input capture for voice translation
- ✅ **Output Routing**: Choose speakers, virtual mic, or both for translated voice output
