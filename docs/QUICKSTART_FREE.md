# Quick Start - FREE VERSION 🆓

**No third-party software required!** This version uses Windows WASAPI directly.

## 1. Install Dependencies
```bash
pip install -r requirements.txt
```

## 2. Run the FREE Version
```bash
python main_tkinter_free.py
```

## 3. Start Translation
1. Click **"Start Translation"**
2. Play audio on your computer (YouTube, games, etc.)
3. Translations will appear!

## That's It! 🎉

No WASAPI loopback needed. No paid software. Just Python libraries!

## If It Doesn't Work

If you get errors, try:
1. Enable "Stereo Mix" in Windows Sound settings → Recording tab
2. Or use the standard version: `python main_tkinter.py` (requires WASAPI loopback)
