## Full Description

### Break the Language Barrier in Competitive Gaming

Communicate with players from around the world! Real-Time Game Voice Translation captures in-game voice chat, transcribes speech in real-time, and translates it to your preferred language with on-screen subtitles.

Works with CS:GO 2, Valorant, Apex Legends, Dota 2, and other competitive multiplayer games.

**Real-Time Translation**

- Captures game voice chat automatically
- Transcribes speech using advanced AI (OpenAI Whisper)
- Translates to 15+ languages instantly (optimized for gaming communities)
- Low latency (<1 second with local models)
- **Works offline** - No internet connection needed after initial setup

**On-Screen Subtitles**

- Customizable subtitle overlay
- Adjustable text size and position
- Always-on-top display
- Clean, non-intrusive design

**Easy Setup**

- Auto-configuration wizard
- Works with VB-Audio Virtual Cable (free)
- Intuitive interface designed for gamers
- Comprehensive documentation

**Professional Quality**

- Hardware-bound license system

- Optimized for performance
- Windows 10/11 compatible
